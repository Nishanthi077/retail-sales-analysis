#  Retail Sales EDA

A concise Exploratory Data Analysis (EDA) on retail sales data to identify key sales trends, customer behavior, and product performance.

##  Highlights
- Data cleaning & preprocessing  
- Summary statistics  
- Sales trend analysis  
- Customer & product insights  
- Visualizations using Matplotlib & Seaborn  

##  Tools
Python 路 Pandas 路 NumPy 路 Matplotlib 路 Seaborn

##  Summary
Identifies top-selling categories, peak sales periods, and customer patterns to support better retail decisions.
